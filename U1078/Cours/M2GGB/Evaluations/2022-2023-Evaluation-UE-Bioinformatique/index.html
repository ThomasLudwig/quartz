# Unix

  3. Vous avez un fichier par gène dans un répertoire. Chaque fichier est nommé nom_du_gene.gene. Il n’y a pas d’en-tête dans le fichier et les lignes du tableaux suivent le format suivant :

# SAM/BAM

  1. Quels sont les fichiers d’entrée nécessaires pour obtenir un alignement BAM/SAM ?
  2. Un read possède le FLAG 99 quel sera vraissemblement le FLAG de son mate ? Expliquez
  3. Expliquer les CIGAR suivants et préciser la longueur du read dans chaque cas:
    • 120M
    • 50M3D47M
    • 140M10S
  4. Ecrire une commande qui compte le nombre de reads avec du hard clipping
  5. Si la référence est CCTCGGTATCTTTCTAGAAGTCCTCGT et que le read est CCCCGGTATCTTGTCTAGAAGTCTCGT, quel sera le CIGAR ?
  6. Pourquoi faut-il trier les fichiers BAM ?
  7. Dans un fichier BAM qui contient plusieurs individus, quel attribut permet de differencier les reads d’un individu donné ?

# VCF

  1. Un fichier VCF posède la ligne suivante, quelles sont les allèles (lettres) de chaque individu ? Combien y a-t’il d’homozygotes ? Qui a une pronfondeur de couverture (depth) de plus de 20 ?

    #CHROMPOSIDREFALT[…]FORMATalicebobcharliedavidevefrank
    chr1245671.CT,A[…]GT:DP:GQ0/0:32:990/1:21:991/1:13:600/2:60:991/1:8:10./.:.:.

  2. Pourquoi serait-il étrange de voir la ligne suivante dans un fichier VCF ?

    #CHROMPOSIDREFALT[…]FORMATalicebobcharliedavidevefrank
    chr1257954.GC[…]GT:DP:GQ0/0:32:990/0:21:99./.:.:.0/0:60:990/0:8:10./.:.:.

  3. Ecrire une commande qui permet de compter le nombre de sites variants dans un fichier VCF

  5. Ecrire une commande qui permet de compter le nombre d’individus dans un fichier VCF. Je vous suggère d’utiliser la commande tr quelque part, mais ce n’est pas obligatoire.

  6. La recalibration d’un fichier VCF par VQSR permet de classer les variants dans des tranches. Pourquoi intégrer (ou non) un nombre important de tranches dans une analyse peut-il être intéressant ?

# Base de Données et Annotations

  1. GnomAD répertorie la fréquence des variants dans plusieurs population. Pourquoi est-il souhaitable de filtrer les variants en fonction de leur fréquence ?
  2. Quelle est la frequence dans FrEx du variant rs192173389.
  3. Ce variant vous semble t-il avoir été bien couvert lors du séquençage ?
  4. Comment interpretez vous ses scores SIFT/Polyphen ?
  5. A quelle classe d’Impact appartient l’annotation missense_variant ? D’après-vous pourquoi est-elle dans cette classe ?
  6. L’acide aminé impacté par le variant rs545014063 vous semble t’il très conservé chez les vertébrés ?
  7. Pour ce même variant, en vous basant sur les annotations CADD, quel allèle est le plus délétère et quel semble être l’allèle ancestral ?

Dans USCS (Genome Human GRCh37/hg19) chercher le variant : rs113993960

  1. Quel gène est impacté
  2. Quels sont les nucléotides et l’acides aminés impactés par ce variant
  3. En survolant l’acide aminé avec votre souris, dans quel exon du gène se trouve cet acide aminé ?
  4. En zoomant en arrière, quels transcrits sont impactés/non-impactés par ce variant. (afficher les données de Ensembl Genes)
  5. En affichant en “Full” les données de OMIM Alleles quel est la conséquence de ce variant sur la protéine. Quelle pathologie peut en découler ?
